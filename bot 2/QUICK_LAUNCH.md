# Quick Launch

Use this file from the project root:

- [RUN_WEBAPP.bat](F:\Action!\Trade\op\New%20folder\RUN_WEBAPP.bat)
  - Starts the backend and the web dashboard.

## Product Direction

This project now focuses on:

- Telegram bot
- Web dashboard
- Telegram Mini App

Desktop and mobile app shells were removed to keep the product focused on the strongest path.

## Notes

- Run the backend first.
- Open the dashboard from your browser:
  - `http://127.0.0.1:8000/`
- Open it from another phone on the same Wi-Fi using your computer IP:
  - `http://192.168.1.50:8000/`
- Add the API token in the URL on first open if needed:
  - `http://192.168.1.50:8000/?api_token=YOUR_TOKEN`
- Do not use `127.0.0.1` on the phone.
